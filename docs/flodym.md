# The flodym library

REMIND-MFA is built on the [flodym](https://flodym.readthedocs.io/) MFA library, which due to its careful API design allows applications to be easily readable and flexibly extensible and changeable. flodym was developed in-house based on the concepts of ODYM [@Pauliuk20]. It offers an [extensive documentation](https://flodym.readthedocs.io/) and has a quickly growing external user base.
